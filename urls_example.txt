# URL文件示例
# 支持多种格式，以 # 开头的行为注释

# 格式1: 每行一个URL（会自动生成考号）
# https://cdn.example.com/image1.png

# 格式2: kaohao URL（推荐）
# 17720059002723 https://cdn.phototourl.com/uploads/2026-02-25-e61cbb21-c8fb-4386-b76b-cc144a6a5eaf.png

# 格式3: kaohao URL filename
# 17720059002723 https://cdn.phototourl.com/uploads/2026-02-25-e61cbb21-c8fb-4386-b76b-cc144a6a5eaf.png image.png

# 格式4: JSON格式（每行一个JSON对象）
# {"kaohao": "17720059002723", "url": "https://cdn.phototourl.com/uploads/2026-02-25-e61cbb21-c8fb-4386-b76b-cc144a6a5eaf.png", "filename": "image.png"}
